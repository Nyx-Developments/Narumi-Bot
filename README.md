# NarumiBot

## Project Structure

```
Narumi.Framework/					# Framework Code
Narumi.Framework.Tests/		# Tests
Narumi.Bot/								# Executable
```

## Compiling and Running

### Compile

```bash
dotnet build
```

### Run

```bash
# Set the TOKEN environment variable
TOKEN = "Your discord bot token"
dotnet run --project Narumi.Bot
```
